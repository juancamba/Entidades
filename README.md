# Entidades

Cuando se lleva a cabo un análisis temporal de muestras que contienen una o varias variables en diversos dominios de aplicación, 
tales como la medicina, la electrónica o la inspección técnica de vehículos, se observa que la estructura utilizada para almacenar 
estas muestras y variables guarda similitudes significativas, independientemente del dominio específico de aplicación.
Al abstraernos de la idiosincrasia de un dominio particular, es posible conceptualizar la entidad a la que pertenece una muestra y 
considerar el dominio de aplicación como un campo de investigación.

Esta abstracción del problema específico del dominio proporciona la oportunidad de desarrollar una aplicación genérica destinada al 
estudio temporal de las variables en las muestras. En otras palabras, la aplicación sería configurada para adaptarse a prácticamente 
cualquier dominio de interés. Este enfoque permite la creación de una herramienta versátil capaz de abordar el seguimiento temporal 
de variables en diversos campos de investigación, ofreciendo una solución configurable y adaptable a una amplia gama de dominios 
de aplicación.
